mac下安装andbug安装

Apr 9, 2015

AndBug是一个针对Android平台的Dalvik虚拟机的调试器, 目标用户是逆向工程师或者开发人员.安装的过程中遇到一些小问题，没有在网上找到答案，记录一下。

版本:10.10.2

pip install bottle

不安装无法使用navi命令

#下载andbug

git clone https://github.com/swdunlop/AndBug.git

在AndBug目录下，make。提示错误。python2 命令不存在。在linux机器上看了一下，确实有python2这个命令，是链接到python的。所以在/usr/bin
目录下建立python2软链接，链接到python

sudo ln -s python python2

之后make就可以了。

然后执行./andbug 报错

Traceback (most recent call last):
  File "/usr/bin/andbug", line 66, in <module>
    main(args)
  File "/usr/bin/andbug", line 28, in main
    import andbug, andbug.cmd, andbug.command
ImportError: No module named andbug

需要在.oh-my-zsh/oh-my-zsh.sh添加或者.bash_profile

export PYTHONPATH=`pwd`/lib

然后把 andbug 复制到/usr/bin 就可以在任意位置使用andbug命令了。
